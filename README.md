# Breakout / fop_17

fop_17 is a Breakout clone in java
It was created during winter terms for FOP by 
Aron Heinecke, Niko Mitura, Tim Jäger & Simon Kohaut

## Building

fop_17 can be imported as eclipse project and run out of the box.
Limitations are multi monitor setups as lwjgl 2.X has it's problems with that.
